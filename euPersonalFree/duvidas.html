Dúvidas

Precisa de realizar algum cadastro?

Para utilizar o sistema eu-Personal gratuito, não há necessidade de realizar cadastro. Poderá utilizar o máximo do sistema sem nenhum pedido de cadastro.

O eu-Personal free é um aplicativo?

O eu-Personal free é um sistema web, para conseguir usar será necessário de um navegador com acesso a internet. Mas foi desenvolvido e pensado para ser utilizado em diversas plataformas e navegadores.

Posso acessar a mesma informação em multiplas plataforma?

Não. O sistema eu-Personal free, utiliza o local storage do seu navegador. Ou seja, não possui um banco de dados no servidor e sim no navegador. Transformando os dados inseridos serem utilizados somente naquele navegador.

Tem o risco de outro usuário acessar meus dados?

Sim. Como esta salvo no local storage do seu navegador, não há nenhuma criptografia ou segurança do seus dados. Podendo ser excluídos, alterados e acessados sem a sua devida permissão.

Como que é feito o cálculo da avaliação física?

O cálculo utilizado para realizar a avaliação física é a medida de 3 dobras pollock. Para ser um recurso rápido e de confiança para a realização da avaliação. Lembre-se de utilizar todos os equipamentos corretamente.

Como montar uma ficha?

Para montar a ficha, não será necessário preencher todas as linhas disponíveis para fianlizar. Possui aquele número de linhas justamente para não ficar incompleto.

Como visualizar a ficha depois de pronta?

Para visualizar a ficha, é acessar a sessão treinos prontos e gerar um arquivo pdf.
